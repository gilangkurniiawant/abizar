## Deploy ke Netlify

### Cara 1: Drag & Drop
1. Buka https://app.netlify.com/drop
2. Drag folder `Abizar` ke browser
3. Done

### Cara 2: Netlify CLI
```
cd C:\Users\gkurn\Documents\Abizar
netlify deploy --prod
```

### Cara 3: Git
1. `git init && git add . && git commit -m "init"`
2. Push ke GitHub
3. Connect repo di Netlify dashboard

Index page menjadi default landing.
